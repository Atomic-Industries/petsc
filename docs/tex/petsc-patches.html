Patches for PETSc (Release 2.0.28)

You can apply all the general patches by obtaining petsc-patches-2.0.28.1.tar.gz and extracting it with 
cd $PETSC_DIR ; gunzip -c petsc-patches-2.0.28.1.tar.gz | tar xf -
Make sure you are in the PETSc directory (for example /home/bsmith/petsc-2.0.28) before you run the tar command.

For Windows NT/95/98 download the patches file petsc-patches-2.0.28.1.exe and rebuild the libraries.
