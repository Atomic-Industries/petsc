A Project in Parallel Reservoir Modeling

Reservoir simulation study, from a report on work by the University of Texas on a Grand Challenge in Petroleum Reservoir Modeling

Description

The goal of this project, funded out of DOE's Advanced Computing Technology Initiative (ACTI), is the development of a Next-Generation Oil Reservoir Simulator using the latest software, computers, algorithms, and simulation technology. This project is joint with the Center for Petroleum and Geosystems Engineering and the Center for Subsurface Modeling at the University of Texas at Austin. See the project's home page for a broader description of the project and current activities.

Argonne's major contributions include software and computing technologies, linear and nonlinear equation solvers technology, parallel grid maniputations, sensitivity analysis tools, and virtual environments technology.

Specific Accomplishments

The project was officially launched in July 1995. Through August 1996, the following Argonne contributions have been made.

  • enhanced our computational differentiation software tools---a critical step for sensitivity analysis, design, and optimization of complex systems, and
  • revised our parallel nonlinear solvers to provide needed support for black oil models
  • designed and implemented efficient, practical, and scalable preconditioners for linear solvers, including two main families, based on
  • global (i.e. parallel) incomplete factorization (ILU and ICC) preconditioners, and
  • overlapping subdomain based preconditioners (i.e. Schwarz domain decomposition)
  • incorporated coding strategies to obtain the best efficiency for each algorithm by taking advantage of computer memory hierarchies, both
  • internally on a processor by using the cache efficiently, (e.g. loop unrolling and using blocked data structures), and
  • among processors by overlapping communication among processors and computation on the processors
  • began to develop flexible code for the time evolution of discrete PDE problems.
