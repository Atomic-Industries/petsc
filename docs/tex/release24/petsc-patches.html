Patches for PETSc (Release 2.0.24)

You can apply all the general patches by obtaining petsc-patches-2.0.24.3.tar.gz and extracting it with 
cd $PETSC_DIR ; gunzip -c petsc-patches.tar.gz | tar xf -
Make sure you are in the PETSc directory (for example /home/bsmith/petsc-2.0.24) before you run the tar command.

The patch file fixes the following:
  • 2.0.24.4 (Apr 8) - updated install.html with Windows NT instructions.
  • 2.0.24.3 (Apr 8) - incomplete SNES MG example
  • 2.0.24.2 (Apr 8) - bug in CG when using KSPSetAvoidNorms()
  • 2.0.24.1 (Apr 8) - error in handling user-set options prefixes on the solvers for each level of multilevel algorithms.
